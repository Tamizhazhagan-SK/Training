## Project Overview
This directory contains Java practice programs organized by day.
It includes basic programs, OOP concepts, exception handling,
streams, and a Contact Management System project.


## Main Focus
- Core Java concepts
- Classes and objects
- Collections and HashMap
- Exception handling
- Menu-driven console applications
